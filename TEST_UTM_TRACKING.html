üìã Testing Instructions

How to Test:

  1. Click any test link above - This will open your website with UTM parameters
  2. Navigate to the Contact page - The UTM data will persist
  3. Fill out the contact form - Include all required fields
  4. Submit the form - The UTM data will be included automatically
  5. Check Firebase - Go to Realtime Database ‚Üí contacts ‚Üí latest entry
  6. Verify the data - Look for utm_source, utm_medium, and source_description fields

What to Look For in Firebase:

  1. utm_source - Should match the source you clicked (whatsapp, facebook, google, etc.)
  2. utm_medium - Should show the medium (social, cpc, email, organic)
  3. utm_campaign - Should show the campaign name (test_whatsapp, test_fb_ads, etc.)
  4. source_description - Human-readable source (e.g., "Facebook Ads", "WhatsApp")
  5. referrer - The page that referred the user
  6. landing_page - The full URL with UTM parameters
